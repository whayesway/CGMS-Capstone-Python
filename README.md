# Montessori Capstone Map — Python Streamlit Version

This is the Python web app version of the Montessori Capstone Map.

It is built with:

- `app.py` — Python Streamlit app
- `requirements.txt` — Python packages for Streamlit Cloud
- `.streamlit/config.toml` — dark theme settings
- `images/` — optional image folder

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud

Use the GitHub file URL for `app.py`, or use the interactive picker and choose:

- Repository: your repo
- Branch: `main`
- Main file path: `app.py`

## Optional images

Add these files inside the `images/` folder if you want the image placeholders to display:

- `four-planes.png`
- `cosmic-education.png`
- `3-part-cards.png`
